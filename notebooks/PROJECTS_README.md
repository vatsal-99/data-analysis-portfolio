## Overview

This repository contains 5 data analysis and machine learning projects demonstrating skills in Python, data visualization, statistical analysis, and predictive modeling. Each project addresses real-world problems using industry-standard techniques and libraries.

## Projects

### 1. 911 Emergency Calls Analysis
**File:** `911_Calls_Analysis.ipynb`  
**Dataset:** Montgomery County, PA 911 Call Data (Kaggle)  
**Objective:** Analyze emergency call patterns and identify trends

**Key Skills:**
- Time series analysis
- Data visualization with Matplotlib and Seaborn
- Pattern recognition in temporal data
- Exploratory data analysis

**Highlights:**
- Analyzed thousands of emergency calls to identify peak hours and seasonal patterns
- Visualized call distribution across different emergency types (EMS, Fire, Traffic)
- Insights help optimize emergency service resource allocation

---

### 2. E-commerce Customer Analysis
**File:** `Ecommerce_Customer_Analysis.ipynb`  
**Dataset:** E-commerce customer behavioral data  
**Objective:** Predict customer lifetime value using linear regression

**Key Skills:**
- Linear regression modeling
- Feature engineering
- Model evaluation (R-squared, RMSE, MAE)
- Business analytics

**Highlights:**
- Built predictive model for customer yearly spending
- Analyzed correlation between time spent on platform and purchase behavior
- Provided actionable recommendations on mobile vs web platform investment

---

### 3. Financial Data Analysis
**File:** `Finance_Data_Analysis.ipynb`  
**Dataset:** Historical stock market data  
**Objective:** Analyze stock trends and calculate financial metrics

**Key Skills:**
- Financial data analysis
- Technical indicators (moving averages, volatility)
- Time series visualization
- Risk and return calculations

**Highlights:**
- Comprehensive analysis of stock price movements
- Volume analysis and trend identification
- Risk assessment through volatility metrics

---

### 4. Housing Price Prediction
**File:** `Housing_Price_Prediction.ipynb`  
**Dataset:** USA Housing dataset  
**Objective:** Predict house prices using multiple linear regression

**Key Skills:**
- Multiple linear regression
- Feature correlation analysis
- Residual analysis
- Model validation

**Highlights:**
- Predicted house prices based on area income, house age, number of rooms, and population
- Strong model performance with high R-squared value
- Real-world application in real estate valuation

---

### 5. Titanic Survival Prediction
**File:** `Titanic_Survival_Prediction.ipynb`  
**Dataset:** Titanic passenger data (Kaggle)  
**Objective:** Classify passenger survival using logistic regression

**Key Skills:**
- Logistic regression (binary classification)
- Data preprocessing and feature engineering
- Handling missing data
- Classification metrics (accuracy, precision, recall)

**Highlights:**
- Built classifier to predict survival with strong accuracy
- Identified key factors: gender, class, age, family size
- Demonstrated complete ML pipeline from data cleaning to evaluation

## How to Use

- Clear problem statement
- Step-by-step analysis
- Visualizations and insights
- Key findings summary

To run these notebooks:
```bash

# Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```
